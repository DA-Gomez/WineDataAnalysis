##Two database needed##
winequality-red.csv
winequality-white.csv

you need to open both of them in a same folder or environment to be able to run the scrips properly.

##how to use##
1.open MATLAB
2.make sure you load winequality-red.csv and winequality-white.csv database with the main code
3.run the main script

##Requirement##
MATLAB R2021a or newer this version recommended.
Statistics and Machine Learning Toolbox Add-Ons require for fitctree, crossval, ...

##How to install the Learning Toolbox Add-Ons##
-you need MATLAB licence
-Go to Home>Add-Ons>Get Add-Ons>search for Statistics and Machine Learning ToolboxAdd-Ons
